# 🐄 Cow Run

An endless runner game built with vanilla JavaScript, HTML5 Canvas, and CSS. Help the cow dodge obstacles and collect salami to rack up the highest score!

## 🎮 Gameplay

- **Jump** using **SPACE** or **TAP** (mobile)
- **Double Jump** for advanced maneuvers
- **Restart** the game with **ENTER**
- Collect salami to increase your score
- Avoid obstacles to keep the cow running

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Press SPACE (or tap on mobile) to make the cow jump
3. Dodge incoming obstacles
4. Collect salami for points
5. Press ENTER to restart when the cow hits an obstacle

## 📁 Project Structure

- **index.html** - Main HTML structure and canvas setup
- **script.js** - Game logic and mechanics
- **style.css** - Styling and animations

## 🛠️ Technologies Used

- **HTML5 Canvas** - Game rendering
- **Vanilla JavaScript** - Game logic and controls
- **CSS3** - Styling and animations
- **Press Start 2P Font** - Retro game aesthetic

## 🎨 Features

- Smooth animations and pixel-perfect collision detection
- Responsive canvas-based rendering
- Retro game styling with Press Start 2P font
- Score tracking system
- Mobile and desktop support
